# Queue Simulation System (Assignment 2)

A multithreaded Java application that simulates customer queues and a scheduler using Java threads and a graphical interface.

## Features
- Thread-based simulation of customers and servers
- Scheduling based on arrival and service times
- Real-time GUI using Java Swing
- Logging output to file

## Technologies
- Java
- Threads
- Java Swing

## How to Run
1. Open the project in IntelliJ or Eclipse.
2. Compile and run the `Main` class to start the simulation.
